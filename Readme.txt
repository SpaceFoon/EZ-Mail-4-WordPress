== Description ==

A plugin to configure SMTP settings and assign different PrivateEmail mailboxes to WooCommerce orders, BuddyPress forum notifications, account-related emails, and admin emails.

== Installation ==
1. Upload the `privateemail-mail-config` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > PrivateEmail Mail Config** to configure your email settings.
